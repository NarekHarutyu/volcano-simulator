# Volcano Formation: From Mantle Melting to Lava Layers

An interactive 2D cross-section simulator that shows how a volcano forms — mantle melting produces magma, magma rises through a conduit, eruptions deposit lava, and repeated eruptions gradually build the volcano cone.

Built for the **EEPS 0010 Talent Showcase** to communicate a geologic feature (a volcano) and the geologic process that creates it in a clear, visual way.
